Быстрый старт

Начните работу с API за несколько минут

1. Аутентификация

curl -X POST https://api.toco.tj/auth \
  -H "Content-Type: application/json" \
  -d '{"api_key": "your_api_key"}'

Получите API ключ в личном кабинете и используйте его для авторизации всех запросов.

2. Пример запроса

curl https://api.toco.tj/vacancies \
  -H "Authorization: Bearer {token}" \
  -H "Accept: application/json"

Все эндпоинты возвращают данные в формате JSON с стандартными HTTP кодами ответов.
