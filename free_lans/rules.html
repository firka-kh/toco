Правила платформы TOCO FreelanceHub

Соблюдение этих правил обеспечивает честную и безопасную работу для всех участников

Общие правила

1. Уважение к участникам

  • • Общайтесь вежливо и профессионально
  • • Не используйте оскорбительные выражения
  • • Уважайте культурные и религиозные различия
  • • Соблюдайте конфиденциальность персональных данных

2. Честность и прозрачность

  • • Предоставляйте достоверную информацию в профиле
  • • Не завышайте свои навыки и опыт
  • • Указывайте реальные сроки выполнения работ
  • • Будьте честны относительно своих возможностей

3. Качество работы

  • • Выполняйте работу в соответствии с техническим заданием
  • • Соблюдайте установленные сроки
  • • Предоставляйте работу надлежащего качества
  • • Своевременно информируйте о возникших проблемах

Правила для фрилансеров

Профиль и портфолио

  • • Загружайте только собственные работы
  • • Указывайте реальные навыки и опыт
  • • Обновляйте информацию при изменениях
  • • Используйте профессиональную фотографию

Отклики на проекты

  • • Внимательно читайте техническое задание
  • • Задавайте уточняющие вопросы
  • • Указывайте реалистичные сроки и цену
  • • Не дублируйте одинаковые отклики

Выполнение работ

  • • Соблюдайте согласованные сроки
  • • Регулярно отчитывайтесь о прогрессе
  • • Вносите правки по замечаниям заказчика
  • • Сдавайте работу в оговоренном формате

Общение с заказчиками

  • • Отвечайте на сообщения в течение 24 часов
  • • Используйте только внутренний чат платформы
  • • Не просите контакты для работы вне платформы
  • • Сохраняйте профессиональный тон общения

Правила для заказчиков

Техническое задание

  • • Подробно описывайте требования к работе
  • • Указывайте реальные сроки выполнения
  • • Прикрепляйте необходимые материалы
  • • Устанавливайте справедливый бюджет

Выбор исполнителя

  • • Изучайте портфолио и отзывы
  • • Задавайте уточняющие вопросы
  • • Не выбирайте только по самой низкой цене
  • • Отвечайте на отклики в разумные сроки

Контроль выполнения

  • • Отслеживайте прогресс выполнения
  • • Своевременно проверяйте промежуточные результаты
  • • Давайте конструктивную обратную связь
  • • Не меняйте требования в процессе работы

Приемка работы

  • • Проверяйте работу в течение 3 дней
  • • Четко формулируйте замечания
  • • Принимайте работу при соответствии ТЗ
  • • Оставляйте честные отзывы

Запрещенные действия

Строго запрещено:

  • • Публикация незаконного контента
  • • Плагиат и нарушение авторских прав
  • • Мошенничество и обман
  • • Создание фейковых аккаунтов
  • • Накрутка рейтинга и отзывов
  • • Спам и назойливая реклама
  • • Переманивание клиентов с платформы

Последствия нарушений:

  • • Предупреждение
  • • Временная блокировка аккаунта
  • • Ограничение функций
  • • Удаление контента
  • • Штрафные санкции
  • • Полная блокировка аккаунта
  • • Обращение в правоохранительные органы

Разрешение споров

Процедура разрешения споров

  1. Попытка самостоятельного решения через переписку
  2. Обращение в службу поддержки с описанием проблемы
  3. Предоставление доказательств (переписка, файлы, скриншоты)
  4. Рассмотрение спора модератором в течение 3 рабочих дней
  5. Принятие решения на основе фактов и правил платформы
  6. Исполнение решения всеми сторонами

Система Safe Deal

Для защиты интересов всех участников действует система безопасных сделок:

  • • Деньги блокируются на счете заказчика до завершения работы
  • • Фрилансер получает оплату только после принятия работы
  • • В случае спора средства остаются заблокированными до решения
  • • Платформа гарантирует честность расчетов

Изменения правил

Платформа оставляет за собой право изменять и дополнять правила. Все изменения публикуются заранее с уведомлением пользователей.
