Описание курса

Полный курс по frontend разработке с изучением HTML, CSS, JavaScript и React. За 3 месяца вы освоите все необходимые технологии для создания современных веб-приложений.

Что вы изучите:

  • HTML5 и семантическая разметка
  • CSS3, Flexbox и Grid
  • JavaScript ES6+ и DOM
  • React и современные библиотеки
  • Работа с API и асинхронное программирование
  • Инструменты разработки и деплой

Программа курса

Модуль 1: Основы веб-разработки (4 недели)

  • Введение в HTML
  • CSS и стилизация
  • Адаптивная верстка
  • Практический проект: Лендинг

Модуль 2: JavaScript (4 недели)

  • Основы JavaScript
  • Работа с DOM
  • Асинхронное программирование
  • Практический проект: Интерактивное приложение

Модуль 3: React (4 недели)

  • Компоненты и состояние
  • Хуки и контекст
  • Роутинг и управление состоянием
  • Финальный проект: SPA приложение
